 # 🛒 VirantCart - Modern E-commerce Web App

VirantCart is a premium and fully responsive e-commerce application built with **React.js**. It features a clean UI, smooth navigation, and efficient state management using Context API. 

---

## ✨ Key Features

* **Dynamic Product Listing:** Fetches real-time data from DummyJSON API.
* **Search & Filter:** Advanced search bar with suggestions and category-based filtering.
* **Shopping Cart:** Users can add/remove items and update quantities with real-time price calculations.
* **Wishlist:** Save favorite items for later viewing.
* **Detailed View:** Specific pages for every product with related item suggestions.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens using **Tailwind CSS**.
* **Global Styling:** Centralized configuration for brand colors and themes.

---

## 🛠️ Tech Stack

* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS
* **Routing:** React Router DOM (v6)
* **State Management:** React Context API
* **Icons:** React Icons (Hi, Fi)
* **Data Source:** DummyJSON API

## 🚀 Getting Started

Follow these steps to run the project locally:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/shivanipatels/virant-cart.git](https://github.com/shivanipatels/virant-cart.git)
    ```

2.  **Navigate to the project folder:**
    ```bash
    cd virant-cart
    ```

3.  **Install dependencies:**
    ```bash
    npm install
    ```

4.  **Run the development server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## 📂 Project Structure

* `src/components`: Reusable UI components like Navbar, Footer, and Hero.
* `src/context`: State management for Cart, Wishlist, and Products.
* `src/pages`: Main pages like Home, Cart, ProductDetail, and Auth.
* `src/utils`: Global configuration and utility constants.
